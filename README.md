## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (vXX+)
- [npm](https://www.npmjs.com/) (vXX+)

### Installation

```bash
git clone https://github.com/himanshujays29/Stayzy.git
cd Stayzy
npm install
```

### Running the App

```bash
npm start
```

The app will be accessible at [http://localhost:8080](http://localhost:3000) by default.
